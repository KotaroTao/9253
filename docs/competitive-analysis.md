# MIERU Clinic 競合・類似ツール調査レポート

## 調査概要

MIERU Clinic（患者体験改善プラットフォーム）と類似するツール・サービスについて、日本市場およびグローバル市場の両面から調査を実施した。

---

## 1. 日本市場の類似ツール

### 1.1 Gyro-n「キキコミ」
- **URL**: https://www.gyro-n.com/seo/hack/clinic_meo/
- **概要**: 特許取得済み（特許第7773815号）のクリニック向け口コミ管理ツール。患者アンケートからAIが口コミ下書きテキストを生成し、投稿のハードルを下げる
- **主な機能**: アンケートベースの口コミ文生成、MEOツール、口コミモニタリング
- **コンプライアンス**: Googleガイドライン・ステマ規制対応を謳う
- **MIERU Clinicとの比較**: 口コミ獲得が主軸。スタッフ別分析やダッシュボードは未提供

### 1.2 Hero Innovation「口コミアンケートツール」
- **URL**: https://clinic-kuchikomi.com/
- **概要**: 医療・歯科特化企業（3,500施設以上）が提供するGoogle口コミ獲得ツール
- **主な機能**: Webアンケート作成・自動集計、QRコード/LINE/メール配信、ネガティブ口コミアラート
- **コンプライアンス**: **レビューゲーティング（高評価者のみGoogleへ誘導）を実施** → Googleポリシー違反の可能性
- **MIERU Clinicとの比較**: 総合マーケティング企業の一機能。患者体験分析やスタッフ別追跡なし

### 1.3 口コミPLUS
- **URL**: 比較記事で参照（https://www.shopowner-support.net/）
- **概要**: QRコードベースの口コミ促進ツール。業種を問わず利用可能
- **主な機能**: QRコードからアンケート→口コミ誘導、ChatGPT口コミ返信生成
- **価格**: **月額10,000円（税抜）**
- **コンプライアンス**: **非準拠** — 「とても満足」「満足」の回答者のみGoogleへ誘導するレビューゲーティングを明示的に実施
- **MIERU Clinicとの比較**: 最も直接的な低価格競合だが、コンプライアンス違反モデル。分析ダッシュボード・スタッフ追跡なし

### 1.4 メルプWEB問診（アンケート・口コミ機能）
- **URL**: https://monshin.melp.life/
- **概要**: Web問診システム（1,200院以上導入）に患者満足度アンケート・Google口コミ誘導機能を追加
- **主な機能**: チャット式問診フォーム（280テンプレート）、メールでの満足度調査送信、電子カルテ連携
- **コンプライアンス**: デフォルト設定で満足度フィルタリングの可能性あり
- **MIERU Clinicとの比較**: 本質的にはWeb問診ツール。患者体験分析やスタッフ追跡の専用機能なし

### 1.5 HIIDEL（ヒイデル）
- **概要**: 日本初のNFCスタンド式口コミ収集ツール。スマホをNFCスタンドにタッチしてアンケート・口コミフローへ
- **主な機能**: NFCスタンド（無料提供）、アンケート→口コミパイプライン
- **MIERU Clinicとの比較**: ハードウェア依存の口コミ収集ツール。分析・ダッシュボード・スタッフ追跡なし

### 1.6 COLOURS（カラーズ）by ドクターブリッジ
- **URL**: https://www.day-surgery.co.jp/colours-vision/
- **概要**: SMS送信で来院後の患者アンケートを実施、満足度が高い患者をGoogle口コミへ誘導
- **MIERU Clinicとの比較**: SMS中心。分析・スタッフ追跡なし。マーケティング会社の一サービス

### 1.7 クチコミマーケ
- **URL**: https://review-system.jp/
- **概要**: Google、Caloo、エキテン、Yahoo! ロコなど複数プラットフォーム対応の口コミ管理ツール
- **MIERU Clinicとの比較**: マルチプラットフォーム対応だが患者体験改善の位置づけなし

### 1.8 医療経営支援事務所 口コミシステム
- **URL**: https://mmso.jp/review_measures
- **概要**: 高評価者のみGoogleマップへ誘導するシステム。医師別満足度調査も提供
- **コンプライアンス**: **レビューゲーティングを設計上実施**
- **実績**: 開院18ヶ月で144件・平均4.9のレビュー獲得（事例）
- **MIERU Clinicとの比較**: 医師別満足度調査はMIERUのスタッフ追跡と類似するが、ゲーティングモデル

### 1.9 AI自動返信 by Medrock
- **URL**: https://medrock.co.jp/navi/review-system/
- **概要**: AI口コミ返信自動化ツール。基本機能無料
- **MIERU Clinicとの比較**: 口コミ返信特化であり、収集・分析とは補完的関係

## 2. グローバル市場の類似ツール

### Tier 1: エンタープライズ患者体験プラットフォーム

#### 2.1 Press Ganey（Qualtricsが67.5億ドルで買収）
- **URL**: https://www.pressganey.com/
- **概要**: 医療患者体験測定の最大手。41,000+組織にサービス提供。55億件の患者エンカウンターデータ
- **主な機能**: AI要約・分析・推奨、HCAHPS調査ベンチマーキング、NLP感情分析（50+言語）、予測分析
- **MIERU Clinicとの比較**: 業界の「ゴールドスタンダード」だが病院・大規模医療システム向け。MIERUは「小規模日本クリニック向けのPress Ganey」と位置づけ可能

#### 2.2 Birdeye
- **URL**: https://birdeye.com/
- **概要**: AI駆動のレピュテーション管理プラットフォーム（200,000+事業者利用）。医療向けソリューションあり
- **価格**: Starter $299/月、Growth $399/月、Dominate（カスタム）
- **MIERU Clinicとの比較**: 汎用レピュテーション管理。日本市場未対応。小規模クリニックには高額

#### 2.3 Swell CX
- **URL**: https://www.swellcx.com/
- **概要**: 医療ファーストの患者体験・レピュテーションプラットフォーム。DSO/MSO向け。Google Private Review APIパートナー
- **主な機能**: EHR/PMS連携の自動口コミ依頼、NPS/CSAT調査、Employee Experience Insights、プロバイダー/施術別パフォーマンス追跡
- **MIERU Clinicとの比較**: **グローバルで最も哲学が近い競合** — 患者体験を主軸にしつつ口コミ管理も提供。ただし多拠点組織（5-500+拠点）向けでEHR連携必須

#### 2.5 NexHealth

- **価格**: $350-$450/月〜

- **MIERU Clinicとの比較**: **グローバルで最も機能が類似** — 満足度調査+口コミ管理+プロバイダー別追跡を組み合わせ。ただし米国市場向け・EHR連携前提

- **MIERU Clinicとの比較**: **機能面で最も類似する汎用プラットフォーム** — QRコード調査+スタッフ追跡+口コミ管理。ただし日本語非対応・クリニック特化ワークフローなし

## 4. 戦略的示唆

### 4.1 日本市場のギャップ
日本の既存ツール（口コミPLUS、Hero Innovation、医療経営支援事務所等）の大半は**レビューゲーティングツール**である。満足度で患者をフィルタリングし、高評価者のみGoogleへ誘導する。この手法はGoogleポリシーおよびステルスマーケティング規制に違反する可能性がある。MIERU Clinicのコンプライアンス重視アプローチは**市場における真の差別化要因**であり、クリニックのリスク軽減に貢献する。

### 4.2 「患者体験改善」という軸の不在
日本市場には**患者体験改善を主軸**に据えたツールが存在しない。既存ツールはすべて「Google口コミを増やす」を前面に出している。MIERU Clinicの患者体験プラットフォームとしてのポジショニングは、Press GaneyのQualtrics買収（67.5億ドル）やPX研究会の活動が示すグローバルトレンドと整合する。

### 4.3 スタッフ別分析の空白
日本の競合の中で**スタッフ別パフォーマンス追跡**を提供するツールは皆無。グローバルではSwell CX、RepuGen、Zonka Feedback等が提供するが、いずれも英語圏向け。

### 4.4 QRコードファーストのシンプルさ
グローバル競合の多くはEHR/PMS連携による調査トリガーを前提とする（技術的セットアップが必要）。MIERU Clinicのラミネートカード＋QRコード方式は**技術連携ゼロで即日導入可能**。日本のクリニックにおけるEMR普及状況のばらつきを考慮すると、大きな優位性。

### 4.5 価格設定の余地
日本の競合（口コミPLUS：月額約1万円）は低価格帯、グローバルツールは$199-$599+/月。MIERU Clinicは日本市場で競争力のある価格でありながら、分析・コンプライアンス・スタッフ追跡で大幅に高い価値を提供する余地がある。

### 4.6 規制の追い風
Googleの口コミポリシー強化、日本の2023年ステマ規制、および歯科クリニックに対する消費者庁の措置事例は、コンプライアンス準拠ツールへの追い風となっている。非準拠ツールを使用するクリニックの法的リスクが高まっている。
